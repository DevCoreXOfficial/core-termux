# OpenClaw

Multi-platform AI assistant aggregator

**Package:** openclaw  
**Author:** DevCoreX  
**Repository:** https://github.com/DevCoreXOfficial/core-termux  
**Official:** https://github.com/openclaw/openclaw  
**Type:** AI assistant aggregator (npm global package)  
**License:** MIT

## Description

OpenClaw is a multi-platform AI assistant aggregator that connects to various AI services including Lark, Nostr, Slack, and WhatsApp. It provides a unified interface for interacting with multiple AI platforms from your terminal.

## Dependencies

- Node.js LTS (nodejs-lts)
- npm
- git, ripgrep
- Additional npm packages: @larksuiteoapi/node-sdk, nostr-tools, @slack/web-api, @whiskeysockets/baileys

## Install

```bash
core install ai --openclaw
```

## Uninstall

```bash
core uninstall ai --openclaw
```

## Update

```bash
core update ai --openclaw
```

## Notes

- Installed as a global npm package: `openclaw`
- Command: `openclaw`
- Installs companion SDK packages for multi-platform support
- Requires configuration for each platform
