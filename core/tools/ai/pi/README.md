## Description

Pi Coding Agent is an AI-powered coding assistant developed by Earendil Works. It provides intelligent code generation, completion, and programming assistance directly in your terminal.

## Dependencies

- Node.js LTS (nodejs-lts)
- npm
- ripgrep, git, fd

## Install

```bash
core install ai --pi
```

## Uninstall

```bash
core uninstall ai --pi
```

## Update

```bash
core update ai --pi
```

## Notes

- Installed as a global npm package: `@earendil-works/pi-coding-agent`
- Command: `pi`
- Uses `--ignore-scripts` flag for Android compatibility
- Requires Node.js LTS (installed automatically if missing)
